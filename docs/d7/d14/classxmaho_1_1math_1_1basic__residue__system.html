Detailed Description

template<typename ValueType, ValueType Modulo>
class xmaho::math::basic_residue_system< ValueType, Modulo >

The loop number under a modulo.

This class doing under the least residue system. The modulo must be a positive number, it mean that equivalent value are positive number.

Template Parameters
ValueTypeThe equivalent type. it must be constexpr.
ModuloThe modulo of residue system.

The explicit cast for value_type.

This cast are explicit. ex)
